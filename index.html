Privacy Policy
Content Shown
Carmel Ministries App contains two types of content.
  • Dynamically Changing Content - Data stored in the cloud and managed by cloud admin, who only have the access to perform create, update and delete operation through the app or cloud console. Ex: The Post Widget and Weekly Events Calender Widget in the Homepage of the app.
  • Static Content - Data stored inside the app, only gets a change in a update if needed. Ex: The Carmel Page in the app.
Data Privacy
Feedback is submitted only if the user manually wishes to submit (not mandatory) through the app that help us to improve the quality of the app.

Other than that no data is taken or stored with or without the knowledge of the user. The App is simply a tool that serves the purpose of displaying the Information about Carmel Ministries to everyone.

User Preferences like dark mode, notifications button status are stored on-device only.
